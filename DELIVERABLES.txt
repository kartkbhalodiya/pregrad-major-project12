HOW TO USE
================================================================================

OPTION 1: Streamlit Web App (Recommended for users)
  1. Install dependencies: pip install -r requirements.txt
  2. Run app: streamlit run app.py
  3. Open browser: http://localhost:8501
  4. Select movie and get recommendations

OPTION 2: Jupyter Notebook (For learning and analysis)
  1. Install dependencies: pip install -r requirements.txt
  2. Run: jupyter notebook notebooks/recommendation_system.ipynb
  3. Execute cells sequentially
  4. Follow explanations and visualizations

OPTION 3: Python Script (For integration/automation)
  1. Install dependencies: pip install -r requirements.txt
  2. Import module: from src.recommendation_engine import NetflixRecommender
  3. Create instance: recommender = NetflixRecommender('data/netflix_sample.csv')
  4. Build model: recommender.build_model()
  5. Get recommendations: recommendations = recommender.get_recommendations('Movie Title', 10)

Solution: python -c "import nltk; nltk.download('stopwords')"
